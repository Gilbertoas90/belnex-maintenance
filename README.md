# About

This repository contains the official maintenance landing page for **BELNEX ENERGY**.

The page is displayed while the main corporate website is under development or undergoing maintenance.

It provides visitors with company information and contact details.

# Installation

Clone the repository

```bash
git clone https://github.com/Gilbertoas90/belnex-maintenance.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# Production

```bash
npm run build
npm start
```
